# 6-type-cipher-methods

Features 6 cipher algorithms used widely nowadays.

This includes: 

  -Hill Cipher
  
  -Transposition Cipher
  
  -Ceasar Cipher
  
  -Playfair Cipher
  
  -Vigenere Cipher
  
  -Railfence Cipher
  
Just input a message, provide a key and press encipher. To decrypt the message, press decipher.
